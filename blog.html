A.What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It defines a tree-like structure of objects that represent the document's structure and content. The DOM provides methods and properties to access and manipulate the document's elements, attributes, and text.

B.How do you select an element from the DOM?

To select an element from the DOM, you can use the following methods:
  • getElementById(id): Selects an element with the specified ID
  • querySelector(selector): Selects the first element that matches the specified selector
  • querySelectorAll(selector): Selects all elements that match the specified selector

C.What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to attach event listeners to a parent element instead of individual child elements. This reduces the number of event listeners and improves performance by avoiding unnecessary event handling.

D.How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can use the following methods:
  • setAttribute(name, value): Adds or updates an attribute of the element
  • removeAttribute(name): Removes an attribute of the element
  • getAttribute(name): Returns the value of the specified attribute
